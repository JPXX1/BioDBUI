addfile

  • addfile(): Promise<void>

  • Handhabt das Hinzufgen einer Datei und verarbeitet sie basierend auf dem enthaltenen Datentyp. Diese Funktion liest die Datei als ArrayBuffer, verarbeitet sie mit der FileReader-API und bestimmt dann den Datentyp in der Datei, um spezifische Importoperationen durchzufhren.

    Die Funktion untersttzt mehrere Importtypen, einschlielich:

    • Phylib-Import
    • Phylib-Bewertungen
    • Perlodes-Import
    • Perlodes-Export
    • Phytosee-Export
    • LLBB_Ilat-Import
    • Phytofluss-Export

    Die Funktion aktualisiert verschiedene UI-Elemente und interne Zustnde basierend auf dem Importtyp.

    Returns Promise<void>

    Ein Versprechen, das aufgelst wird, wenn die Dateiverarbeitung abgeschlossen ist.
