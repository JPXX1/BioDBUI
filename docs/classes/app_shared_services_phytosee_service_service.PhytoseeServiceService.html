Phytoflussexport

  • Phytoflussexport(workbook, valspalten, tab, verfahrennr): Promise<void>

  • Exportiert Daten aus einem Workbook in ein spezifisches Format.

    Parameters

    • workbook: any

      Das Workbook-Objekt, das die zu exportierenden Daten enthält.

    • valspalten: any

      Ein Array von Objekten, die die zu verarbeitenden Spalten repräsentieren.

    • tab: any

      Die Tab-Nummer, die verarbeitet werden soll.

    • verfahrennr: number

      Die Verfahrensnummer, um die Spalten zu filtern.

    Returns Promise<void>

    Ein Promise, das sich auflöst, wenn der Exportprozess abgeschlossen ist.

    Bemerkungen

    Diese Funktion verarbeitet die Workbook-Daten, indem sie die Spalten anhand der angegebenen Verfahrensnummer und des Tabs filtert. Anschließend wird das relevante Arbeitsblatt in JSON-Format konvertiert und jede Zeile wird bearbeitet, um Daten zu extrahieren und zu transformieren. Die transformierten Daten werden im messstellenImp-Array gespeichert, und verschiedene Servicemethoden werden aufgerufen, um die Daten zu verarbeiten.

    Beispiel

    const workbook = ...; // Workbook laden oder erstellen
    const valspalten = [...]; // Zu verarbeitende Spalten definieren
    const tab = 1; // Tab-Nummer angeben
    const verfahrennr = 123; // Verfahrensnummer angeben

    await Phytoflussexport(workbook, valspalten, tab, verfahrennr);

Phytoseeexport

  • Phytoseeexport(workbook, valspalten, tab, verfahrennr): Promise<void>

  • Exportiert Daten aus einem Workbook in ein spezifisches Format für Phytosee.

    Parameters

    • workbook: any

      Das Workbook-Objekt, das die zu exportierenden Daten enthält.

    • valspalten: any

      Ein Array von Objekten, die die zu verarbeitenden Spalten repräsentieren.

    • tab: any

      Der Tab-Identifikator für die Daten.

    • verfahrennr: number

      Die Verfahrensnummer, um die Spalten zu filtern.

    Returns Promise<void>

    Ein Promise, das sich auflöst, wenn der Exportprozess abgeschlossen ist.

    Bemerkungen

    Diese Funktion verarbeitet die Workbook-Daten, indem sie die Spalten anhand der angegebenen Verfahrensnummer und des Tab-Identifikators filtert. Anschließend wird das relevante Arbeitsblatt in JSON-Format konvertiert und jede Zeile wird bearbeitet, um Daten zu extrahieren und zu transformieren. Die transformierten Daten werden im messstellenImp-Array gespeichert und der xlsxImportPhylibService.messstellenImp-Eigenschaft zugewiesen.

    Die Funktion behandelt auch spezifische Transformationen für bestimmte Parameter-IDs und verwaltet die Übersicht über Fehler und Importstatus.

    Beispiel

    const workbook = ...; // Workbook laden oder erstellen
    const valspalten = [...]; // Zu verarbeitende Spalten definieren
    const tab = 1; // Tab-Identifikator angeben
    const verfahrennr = 123; // Verfahrensnummer angeben

    await Phytoseeexport(workbook, valspalten, tab, verfahrennr);

  • Importiert Daten aus einem Excel-Arbeitsbuch in das Phytosee-System.

    • verfahrennr: number

    Returns Promise<string>

    Throws

    Wird einen Fehler auslösen, wenn es Probleme mit den Daten gibt, wie z.B. ungültige Daten oder falsche Spaltenüberschriften.
