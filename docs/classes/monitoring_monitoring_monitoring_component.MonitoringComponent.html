filtertaxadaten

  • filtertaxadaten(komp): void

  • Filtert taxonomische Daten basierend auf der angegebenen Komponentennummer.

    Parameters

    • komp: number

      Die Komponentennummer, die verwendet wird, um zu bestimmen, welche taxonomischen Daten gefiltert werden sollen.

    Returns void

    Remarks

    Die Methode verwendet die Funktionen anzeigeBewertungMPService.FilterRichtigesArray, um die Daten zu filtern. und reduceArray, um die Anzahl der Elemente im Array zu reduzieren.

    Example

    this.filtertaxadaten(1);

reduceArray

  • reduceArray(mstTaxaAllg): MstMakrophyten[]

  • Reduziert die Größe des angegebenen Arrays von MstMakrophyten-Objekten auf maximal 2500 Elemente. Wenn die Array-Länge 2500 überschreitet, wird eine Nachricht in der InfoBox angezeigt, die auf die Reduzierung hinweist.

    Parameters

    • mstTaxaAllg: MstMakrophyten[]

      Das zu reduzierende Array von MstMakrophyten-Objekten.

    Returns MstMakrophyten[]

    Ein neues Array, das bis zu 2500 Elemente aus dem ursprünglichen Array enthält.
